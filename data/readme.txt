The data is available at http://neurotycho.org/food-tracking-task (subdural)
To reproduce results from the paper, download
20090611S1_FTT_A_ZenasChao_csv_ECoG32-Motion12
20090121S1_FTT_A_ZenasChao_csv_ECoG32-Motion10

and unpack them. Browse down the unpacked folders until you reach folder named the same as the dataset (i.e. 20090611S1_FTT_A_ZenasChao_csv_ECoG32-Motion12. There should be Motion.csv and ECoG.csv files inside). Move this ("20090611S1_FTT_A_ZenasChao_csv_ECoG32-Motion12") folder here.

For more information on the data, refer to
Chao ZC, Nagasaka Y, Fujii N (2010). "Long-term asynchronous decoding of arm motion using electrocorticographic signals in monkeys." Frontiers in Neuroengineering 3:3. doi:10.3389/fneng.2010.00003
